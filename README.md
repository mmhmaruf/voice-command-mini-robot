# 🤖 Voice Command Smart Robot (TinyML + ESP32)

## 🚀 Overview
Offline AI robot that responds to voice commands.

## 🔥 Features
- Voice control (forward, left, stop)
- TinyML model (Edge Impulse)
- Works offline
- Real-time motor control

## 🧰 Hardware
- ESP32
- INMP441 Microphone
- L298N Motor Driver
